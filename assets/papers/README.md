## Summary

We have added representative images for 43 papers across all categories. Most images are directly linked from official project pages or GitHub repositories, ensuring they stay up-to-date. Only 5 papers still need local images to be extracted from their arXiv PDFs.

## Image Guidelines

1. **Resolution**: Prefer high-quality images (at least 1200px width)
2. **Format**: PNG format for best quality
3. **Content**: Should clearly show the main contribution/architecture
4. **Source**: From paper's main figure or project page

## How to Add Images

1. Download the figure from the paper or project page
2. Rename to match the filename pattern above
3. Place in this directory
4. The markdown files will automatically display them

## Alternative Sources

If official figures are not available:
- Check the paper's arXiv page
- Visit the project website
- Look for presentation slides
- Extract from the PDF (ensure good quality)
